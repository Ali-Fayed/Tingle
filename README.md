## App Layers: 

- Domain Layer = Entities + Use Cases + Repositories Interfaces
- Data Layer = Repositories Implementations + Remote (API) + Data Model
- Presentation Layer (MVVM) = ViewController - ViewModels - Views ( Custom Cells, Headers, etc ..)
